使用指南

  1. 下载汉化助手 zip
  2. 解压汉化助手 zip 到任意文件夹
  3. 运行 EZMaple.exe，注册、登录自己的账号
  4. 购买、激活授权
  5. 选择游戏目录（GMS 是 appdata 文件夹，其他服请选择 MapleStory.exe 所在的文件夹）
  6. 选择汉化选项（仅限岛1）
  7. 点“汉化”按钮，稍等片刻
  8. 提示『汉化完成，请登录游戏体验』之后，就完成了。
