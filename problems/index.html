已知问题 & 常见问题

2017-11-08

如果发现有 BUG、或者有任何意见或建议，欢迎留言、贴吧、Facebook小组、QQ群（244572576）联系。谢谢！
目前更新会比较频繁，请随时留意网站信息。

已知问题

  1. 商城道具名字部分乱码（无解决方法）。
  2. 由于汉化数据主要来源于 CMS，难免有些道具的描述不符合 GMS 的实际情况，以后会慢慢修正。
    • 潜在能力的等级（B/A/S/SS => R/E/U/L）
    • 希纳斯女皇掉落的用于兑换 lv.140 狮心装备的碎片需要 85 片
    • 击败火狼后获得的猎人包，奖励描述错误
    • …… 等等。
  3. 部分的描述文字不换行，导致溢出。
  4. 部分任务的分类错误（任务界面里）。

常见问题

1. 问：为什么有些内容不汉化？

答：服务器上的内容无法汉化、写在主程序（MapleStory.exe）里的内容无法汉化。

2. 问：汉化时出现错误提示，“对 XXX 路径的访问被拒绝。”？

答1：把汉化程序移到其他盘试试。
答2：或者 点右键“以管理员身份运行”。

3. 问：为什么汉化后会出现乱码？

答1：如果是非简体中文系统玩GMS，解决方法请参考：乱码问题修复
答2：如果是中文系统用 Locale Emulator 玩 KMS/JMS 等需要更改系统语言的服务器，可能会出现部分乱码，目前无解，因为这是系统语言问题，汉化程序只是修改游戏数据而已。不便请谅解。

4. 问：汉化时出现以下错误提示之一：
  1. “Method not found: ‘Int32 System.Runtime.InteropServices.Marshal.SizeOf()’”
  2. “Could not load type ‘System.Runtime.CompilerServices.IAsyncStateMachine’…”

答：系统缺少运行汉化程序的必要组件，请到微软下载中心下载并安装该组件，再重新运行汉化程序即可。

5. 问：程序无法运行，双击没反应，管理员身份也没用，怎么都打不开：

答：同 #4，系统缺少运行汉化程序的必要组件。

6. 问：汉化时出现错误提示：

“Error with game version hash : The specified game … version itself.”

答：操作如下：

  1. 按快捷键 + R 打开“运行”窗口；
  2. 输入secpol.msc，点确认；
  3. 打开『本地安全策略』窗口；
  4. 在左边选择『本地策略 - 安全选项』；
  5. 在右边找到『系统加密：将 FIPS 兼容算法用于加密、哈希和签名』；
  6. 双击进入『属性』页，设为【已禁用】，确定；
  7. 重新运行汉化程序即可；
  8. 正常汉化！

解释：

  • 这个错误只会出现在非微软官方渠道获取的系统中，比如 Ghost 类的“雨林木风”、“深度技术”、“电脑城一键装机”等。
    在正常的 Windows 系统中，该项默认都是“禁用”的。
  • 由于游戏数据是加密的，所以汉化过程中，使用了系统的一些加密算法来处理。
    如果开启了，会导致加密算法“跑偏”，达不到预先的结果，从而出现错误。
