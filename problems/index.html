常见问题

问：为什么有些内容不汉化？

答：服务器上的内容无法汉化（比如剧情、NPC 对话）；写在主程序（MapleStory.exe）里的内容无法汉化（比如角色职业、装备基础属性）。

问：提示汉化成功，进游戏却无汉化效果：

解决方法：重启汉化助手。

问：汉化后，无法运行游戏：

解决方法：先还原，再汉化。

问：为什么汉化后会出现乱码？

答1：如果是非简体中文系统玩GMS，解决方法请参考：乱码问题修复
答2：如果你玩的是 韩服（KMS）、日服（JMS）等，很抱歉，无解。

  • 简单来说，就是因为韩国(曾经)使用的汉字，跟我们现在用的汉字(字形&编码)不一样，在我们现在使用的中文(包括繁体)和韩国汉字的某些字没有一一对应的关系，所以导致了乱码。如果有兴趣的话，可以看看 朝鲜汉字(维基百科)。
  • 无法理解为什么自己的电脑是中文的，却不能显示中文？因为 MapleStory.exe 用的是韩文的字符集啊，要不然为什么要用 Locale-Emulator 来运行？就是因为它不支持（我们用的）中文。
  • 日服同理。
问：汉化时出现以下错误提示之一（中英文对照）：
  • “找不到方法: ‘Int32 System.Runtime.InteropServices.Marshal.SizeOf()’”
    “Method not found: ‘Int32 System.Runtime.InteropServices.Marshal.SizeOf()’”
  • “无法加载类型 ‘System.Runtime.CompilerServices.IAsyncStateMachine’…”
    “Could not load type ‘System.Runtime.CompilerServices.IAsyncStateMachine’…”
  • “‘System.Runtime, Version=4.0.0.0, Culture=…’ or one of its dependencies. 系统找不到指定的文件。”
    “‘System.Runtime, Version=4.0.0.0, Culture=…’ or one of its dependencies. The system cannot find the file specified.”

答：系统缺少运行汉化程序的必要组件，请点我打开微软下载中心下载并安装该组件，再重新运行汉化程序即可。

问：程序无法运行，双击提示下图错误：

答：同 #4，系统缺少运行汉化程序的必要组件。

问：登录时出现错误：

答：请稍后再试，或者使用『绑定电脑』方式登录。（绑定电脑后，只能在此电脑登录，可以随时找管理员解除）

问：汉化时出现错误：
  • “系统找不到指定的文件”
  • “The system cannot find the file specified”

答：请检查汉化助手是否缺少文件（比如 aria2c.exe），如果缺少了，请重新解压一下。

问：汉化时出现错误：
  • “对路径“D:\<游戏路径>\shfolder.dll”的访问被拒绝。”
  • “Access to the path “D:\<游戏路径>\shfolder.dll” is denied.”

答：如果你使用了360，有可能会弹框阻止汉化程序的读取该文件（也有可能不弹框），请添加白名单，或者临时关闭360。

问：汉化时出现错误（仅限 MapleRoyals 私服）：

答：MapleRoyals 私服，自2018-12-13版本之后，需要替换一个文件，才可以成功汉化。【点我下载 String.wz】，下载这个文件之后，直接覆盖到游戏文件夹里面的String.wz，然后重新汉化就好了。

问：汉化后进游戏出现 C++ 错误，不汉化进游戏正常：

答：先还原汉化，然后修复游戏，再重新汉化。

P.S. 修复游戏方法：
Nexon Launcher： 在 Play 按钮右上角有个 小齿轮，点进去，然后点 Repair Installation 。
Steam： 右键游戏名称，点属性，里面有 检查游戏完整性 的按钮（因为我没用过steam，但是大概就是类似的操作）。

问：汉化时出现以下错误提示之一：

“此实现不是 Windows 平台 FIPS 验证的加密算法的一部分。”
“Error with game version hash : The specified game … version itself.”

答：操作如下：

方法一（推荐）：

  1. 按快捷键 + R 打开“运行”窗口；
  2. 输入secpol.msc，点确认；（如果提示错误，看方法二。）
  3. 打开『本地安全策略』窗口；
  4. 在左边选择『本地策略 - 安全选项』；
  5. 在右边找到『系统加密：将 FIPS 兼容算法用于加密、哈希和签名』；
  6. 双击进入『属性』页，设为【已禁用】，确定；
  7. 重新运行汉化程序即可；
  8. 正常汉化！

方法二：

  1. 按快捷键 + R 打开“运行”窗口；
  2. 输入regedit，点确认；
  3. 依次展开： HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Lsa\FipsAlgorithmPolicy；
  4. 把 Enabled 的值改为 0 ；
  5. 重新运行汉化程序即可；
  6. 正常汉化！

解释：

  • 这个错误只会出现在非微软官方渠道获取的系统中，比如 Ghost 类的“雨林木风”、“深度技术”、“电脑城一键装机”等。
    在正常的 Windows 系统中，该项默认都是“禁用”的。
  • 由于游戏数据是加密的，所以汉化过程中，使用了系统的一些加密算法来处理。
    如果开启了，会导致加密算法“跑偏”，达不到预先的结果，从而出现错误。
