2017-10-18

中文输入器

功能

  • 用中文聊天（除了“所有人”之外的，组队、好友、工会、私聊都可以）
  • 用中文刷喇叭（可装逼）
  • 用中文搜索物品、任务、地图、怪物（前提是使用了汉化）
  • 用中文给召唤怪（Familiar）、宠物、机器人起名字
  • 用中文创建新角色（可以，但是有条件￥2.00/个）

用法

  1. 在系统设置中添加一个英文的输入法（必须是系统自带的输入法，不能是搜狗切换到英文状态），点我看教程(还没写😂)
  2. 在【中文字符】输入中文
  3. 切换到添加的英文输入法
  4. (a) 一个字符一个字符地复制【拉丁字符】中的“乱码”（必须一个一个复制，不能一次性复制，两个也不行。确实有点麻烦）
    (b) 为了方便复制，添加了一个【复制下一个字符】按钮，点一下就自动复制一个字符，避免选错
  5. 在游戏中 Ctrl + V 粘贴字符
  6. 重复 4 和 5 。
温馨提示：

一个中文等于两个符号，粘贴第一个字符的时候，你看到的是一个【?】问号，不用管的，接着粘贴第二个字符时，你就会看到一个中文字了。

特别说明

因为系统编码的原因，能看到中文的玩家必须使用的是中文版的 Windows 系统，如果对方用的不是中文版的，他看到的就是【拉丁字符】中显示的内容（挺恶心的乱码）。
所以，这个程序会一定程度地 降低 非中文系统的其他玩家的游戏体验。
因此呢，为了防止滥用，这个程序不会免费公开。一人一机一授权。
