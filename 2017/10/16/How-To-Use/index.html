使用方法

2017-10-16 阅读量

使用日期更新到 2017/11/28。

  1. 下载 程序和数据包
  2. 分别解压程序和数据包
  3. 选择游戏目录（GMS 是 appdata 文件夹）以及数据包
  4. 备份游戏数据（不要碰备份后的数据，备份和还原必须通过程序进行操作！）
  5. 选择汉化选项（目前即使全部取消☑，仍然会汉化部分内容）
  6. 点“汉化”按钮，稍等片刻（我测试大概是1分钟左右，视电脑性能而定）
  7. Enjoy it!

注意：进入游戏弹出的活动窗口，内容我核对过了，没问题，但时间以官网为准，一般都是本周末。
