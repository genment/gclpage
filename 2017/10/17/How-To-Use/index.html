使用方法

2017-10-17 阅读量
  1. 下载 程序和数据包
  2. 分别解压程序和数据包
  3. 运行程序，注册、登录自己的账号
  4. 购买、激活授权
  5. 选择游戏目录（GMS 是 appdata 文件夹）以及数据包
  6. 选择汉化选项（目前即使全部取消☑，仍然会汉化部分内容）
  7. 点“汉化”按钮，稍等片刻（我测试大概是1分钟左右，视电脑性能而定）
  8. Enjoy it!
