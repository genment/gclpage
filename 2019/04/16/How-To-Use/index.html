使用方法

2019-04-16 阅读量

┌┬────────────────────────┬┐
││ 这是一个充满爱心但又有点啰嗦的比较有用的教程 ││
└┴────────────────────────┴┘

  1. 双击那个蓝色的、图标很丑的那个应用程序。

  2. 如果你看到一个小小的蓝蓝的东西在转圈圈，请直接跳到第8步。

  3. 如果双击后，不能打开，或者出错了，这个时候，你应该再双击那个【win7】开头的txt文件。

  4. 它里面有一个下载链接，复制这个链接，然后打开你心爱的浏览器，在地址栏中Ctrl+V粘贴你刚刚复制的链接上去，按下你的键盘中那个大大的会拐弯的上面写着ENTER的键，就会打开一个页面。

  5. 打开网页后，什么都不用做！等几秒钟，就会自动开始下载一个叫做 .net framework 4.7.2 的东西。不用管它是什么，我不会害你电脑中毒的，请放十万八千个心。如果你实在好奇，就去百度一下吧。

  6. 下载完成后，你自己安装一下吧。（我相信你知道怎么安装软件的）

  7. 安装完后，请回到第1步。

  8. 如果你第一次用我们的汉化（我猜你是第一次），你需要先注册一个账号。这个账号不是你的游戏账号，是汉化账号，用来管理你的汉化的。注册成功后，登录你的邮箱，你就会看到有一封验证汉化账号的邮件，点开它，再点开里面的链接。登登登登！验证成功！

  9. 回到汉化登录窗口，输入你那不可告人的密码，就可以登录成功了！

  10. 第一次登录的时候，会问你是否绑定电脑，你自己看着办吧。如果你需要在网吧玩，你就直接勾选密码输入框下方的【拒绝绑定电脑】，就不会问你了。

  11. 终于进来了！进来之后，会有个弹框，引导你去激活你的汉化账号，点一下那个蓝蓝的椭圆的【确认】，切换到【VIP会员】页面。

  12. 【VIP会员】页面有详细的支付、激活教程

  13. 【支付宝支付】支付完后，打开你的交易详情，里面有个叫做订单号的，31位数字，这！就是你的激活码！

  14. 【微信支付】支付完后，点开你的账单详情（页面是黄的，有支付金额、时间、单号的那个页面），截个图，加个QQ群：244572576，找那个名字很长，写着“有事找我”的管理员。把截图发给他就可以了，（12小时内）他看见了自然会给你订单号（激活码）。

  1. 激活成功后，你可以轻轻抓住你的鼠标，慢慢移动到左边的【选项】那里，轻轻地点一下鼠标左键，在这里设置你想要的，和不想要的，感兴趣的，和不感兴趣的汉化内容。

  2. 设置好后，回到汉化页面，点开始汉化！！！

  3. 耐心点，等一会儿，去洗个手，上个厕所，喝口水，到外面看看风景，回来就好了。

  4. 如果不幸，出现了一点点小情况，去这里吧： https://hanhua.tk/problems
